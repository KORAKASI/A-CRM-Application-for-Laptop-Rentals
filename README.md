# 💻 Laptop Rental CRM Application (Salesforce)

This project is a Salesforce-based CRM application designed to manage laptop rental operations, track asset allocation, and automate communication with laptop owners.

---

## 🚀 Project Overview

The system helps streamline the process of renting laptops by:
- Tracking laptop availability and rental status
- Managing customer and rental records
- Automating email notifications to laptop owners

This solution improves operational efficiency and reduces manual tracking efforts.

---

## 🎯 Objectives

- Build a centralized system to manage laptop rentals
- Track allocation and return status of laptops
- Automate email notifications to owners
- Improve visibility and control over assets

---

## 🛠️ Technologies Used

- Salesforce Platform
- Custom Objects & Fields
- Process Builder / Flow Automation
- Email Alerts
- Reports & Dashboards

---

## 🧩 Key Features

- 📦 Laptop Inventory Management  
- 👤 Customer & Rental Tracking  
- 🔄 Status Tracking (Available / Rented / Returned)  
- 📧 Automated Email Notifications to Owners  
- 📊 Reports and Dashboards for Monitoring  

---

## ⚙️ System Workflow

1. Laptop is added to the system with details
2. Customer requests a rental
3. Laptop is assigned and status is updated to "Rented"
4. Automated email is sent to the owner
5. Upon return, status is updated to "Available"

---

## 📊 Business Value

- Reduces manual tracking errors  
- Improves asset utilization  
- Enhances communication through automation  
- Provides real-time visibility of rental status
